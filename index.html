Augmented Nature UniProt 26 AI-ready protein tools (orthologs, structures, pathways); ask detailed UniProt queries via chat. UniProt Core DB
Augmented Nature Ensembl 25 tools for gene, variant & comparative genomics; e.g. “get FASTA for human BRCA1”. Ensembl Core DB

gget-mcp Swiss-army genomics toolkit (BLAST, ARCHS4, COSMIC); single-line RNA-seq QC. Mixed (NCBI, ARCHS4, COSMIC…) Analysis Tool
opengenes-mcp Natural-language longevity-gene queries; e.g. “find CSF1R variants extending lifespan”. OpenGenes Specialized DB

synergy-age-mcp Query synergistic gene–longevity interactions for combinatorial therapy design. SynergyAge Specialized DB
bio-mcp-blast Run BLAST locally/cloud via AI; supports custom DB creation. NCBI BLAST Analysis Tool
bio-mcp-bwa AI-triggered read alignment (BWA-MEM) to reference genomes. BWA Analysis Tool
bio-mcp-samtools AI-driven SAM/BAM manipulation (sort, index, stats). SAMtools Analysis Tool
bio-mcp-seqkit Ultrafast FASTA/FASTQ manipulation; great for pre-alignment QC. SeqKit Analysis Tool
bio-mcp-queue Redis-Celery job queue for long-running tasks; plug other Bio-MCP tools in. Redis / Celery Infrastructure
gnomAD MCP Query population allele frequencies & variant annotations across gnomAD versions. gnomAD Specialized DB
GWAS Catalog MCP Instantly pull published GWAS hits for any trait or gene. GWAS Catalog Specialized DB
BioGRID MCP AI queries of protein & genetic interaction networks. BioGRID Specialized DB
PubMed MCP Natural-language literature mining — fetch abstracts, MeSH terms, citation counts. PubMed Literature/Clinical
ClinicalTrials.gov MCP Search trials by drug, condition, location; keeps conversational context. ClinicalTrials.gov Literature/Clinical
PubTator MCP Pre-annotated biomedical entities (genes, diseases, chemicals) from PubMed texts. PubTator Central Literature/Clinical
bioRxiv MCP Monitor latest preprints; AI alerts for new papers matching given topics. bioRxiv & medRxiv Literature/Clinical
DrugBank MCP Deep drug & interaction look-ups (requires DrugBank API key). DrugBank Specialized DB
OpenTargets MCP AI-driven target ID & prioritization using genetics & pathway evidence. OpenTargets Specialized DB
Galaxy MCP AI control of Galaxy workflows, tools, histories — e.g. “run RNA-seq pipeline on dataset X”. Galaxy Platform Workflow
BioConda MCP AI installs bio-software on-the-fly from BioConda. BioConda Package Manager
BioNext MCP High-level workflow orchestrator chaining SeqKit → BWA → SAMtools automatically. Bio-MCP ecosystem Workflow
NCBI MCP Server Agentic access to the full NCBI suite: PubMed, GenBank, BLAST, Assembly, etc. NCBI E-utilities & BLAST Analysis Tool
GEO MCP Autonomous GEO dataset discovery; iteratively refines queries when initial search fails. NCBI GEO Specialized DB
PLSDB MCP Search plasmid sequences & metadata; useful for tracking AMR plasmids. PLSDB Specialized DB
